================================================================================
DEVELOPMENT WORKFLOW
================================================================================

To customize and extend:

1. ADD A NEW STAT:
   - Edit STAT_CATEGORIES in lib/constants.js
   - Add computation in StatCalculator.js computedStats useMemo
   - Auto-renders in correct section

2. CUSTOMIZE A GEAR SLOT:
   - Edit GEAR_SLOTS in data/gear-slots.js
   - Change x/y percentages, label, type filter

3. ADD A NEW CLASS:
   - Edit CLASSES in data/class-data.js
   - Auto-available in BuildPlanner class picker

4. CHANGE COLORS/FONTS:
   - Edit DESIGN and RARITY_COLORS in lib/constants.js
   - All components auto-use updated values

5. ADD ITEM AFFIXES/STAT SCALING:
   - Fetch d4_affixes in StatCalculator
   - Apply multipliers based on rarity tier
   - Add to computed stats

6. SAVE/EXPORT BUILDS:

   - POST character state to Supabase
   - Return build ID for sharing

   - Sync state across users
